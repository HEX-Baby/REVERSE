# Base64

将输入的每$3个字节 = 24bits$ 按 每$6bits$分成一组,变成$4$个小于$64$的索引值

### 识别
然后通过一个索引表得到4个可见字符，$通常而言 Base64 的识别特征为索引表$

传统的索引表：
ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/

当然还有变种，区别在于改变索引表。
解密的时候，如果索引表不是原来的话，就要先把密文按索引表相应位置转换为$标准索引表的密文$

### 加密
```cpp
// Base64 字符集
static const string base64_chars =
"ABCDEFGHIJKLMNOPQRSTUVWXYZ"
"abcdefghijklmnopqrstuvwxyz"
"0123456789+/";

string base64_encode(const vector<unsigned char>& input_data) 
{
    string encoded_string;
    encoded_string.reserve(((input_data.size() / 3) + (input_data.size() % 3 > 0)) * 4);
    int val = 0, valb = -6;
    for (unsigned char c : input_data) 
    {
        val = (val << 8) + c;
        valb += 8;
        while (valb >= 0) 
        {
            encoded_string.push_back(base64_chars[(val >> valb) & 0x3F]);
            valb -= 6;
        }
    }
    if (valb > -6) 
    {
        encoded_string.push_back(base64_chars[((val << 8) >> (valb + 8)) & 0x3F]);
    }
    while (encoded_string.size() % 4 != 0) 
    {
        encoded_string.push_back('=');
    }
    return encoded_string;
}

string base64_encode(const string& input_string) 
{
    vector<unsigned char> input_data(input_string.begin(), input_string.end());
    return base64_encode(input_data);
}
```

# TEA

TEA算法是一种分组加密算法，密钥为$128bits$, 明文为 $64bits$，主要做了32轮变换，每轮变换都涉及移位和变换。

### 识别TEA
TEA算法中有一个固定的常数$0x9e3779b9$ 或者$0x61c88647$

### 加密
```cpp
// TEA 使用的 delta 常量（推荐值）
const uint32_t DELTA = 0x9e3779b9;
// TEA 加密轮数（通常为 32 轮）
const int ROUNDS = 32;
// 加密函数
void tea_encrypt(uint32_t v[2], const uint32_t key[4]) 
{
    uint32_t v0 = v[0], v1 = v[1];
    uint32_t sum = 0;
    for (int i = 0; i < ROUNDS; ++i) 
    {
        sum += DELTA;
        v0 += ((v1 << 4) + key[0]) ^ (v1 + sum) ^ ((v1 >> 5) + key[1]);
        v1 += ((v0 << 4) + key[2]) ^ (v0 + sum) ^ ((v0 >> 5) + key[3]);
    }
    v[0] = v0;
    v[1] = v1;
}
// 解密函数
void tea_decrypt(uint32_t v[2], const uint32_t key[4]) 
{
    uint32_t v0 = v[0], v1 = v[1];
    uint32_t sum = DELTA * ROUNDS;
    for (int i = 0; i < ROUNDS; ++i) 
    {
        v1 -= ((v0 << 4) + key[2]) ^ (v0 + sum) ^ ((v0 >> 5) + key[3]);
        v0 -= ((v1 << 4) + key[0]) ^ (v1 + sum) ^ ((v1 >> 5) + key[1]);
        sum -= DELTA;
    }
    v[0] = v0;
    v[1] = v1;
}
```
### 解密
```py
def tea_decode(cipher, key, delta):
    v0, v1 = cipher
    k0, k1, k2, k3 = key
    sum = (delta * 32) & 0xFFFFFFFF

    for i in range(32):
        v1 = (v1 - (((v0 << 4) + k2) ^ (v0 + sum) ^ ((v0 >> 5) + k3))) & 0xFFFFFFFF
        v0 = (v0 - (((v1 << 4) + k0) ^ (v1 + sum) ^ ((v1 >> 5) + k1))) & 0xFFFFFFFF
        sum = (sum - delta) & 0xFFFFFFFF
```

# AES

加密过程：

轮密钥加 ----> 
9轮循环代换（1-字节代换； 2-行移位； 3-列混淆； 4-轮密钥加） ---->
最终轮（1-字节代换； 2-行移位； 3-轮密钥加）
